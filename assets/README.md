# Assets Directory

This directory contains media assets for the GrOMP GitHub Pages website.

## Required Assets

The following assets should be placed in this directory:

### Main Media Section
- `teaser_image.png` - Main teaser image for the paper
- `compilation2x.mp4` - Main teaser video for the paper

### Task 1: Nut Threading
- `nut_dp.mp4` - vanilla DP nut trial compilation
- `nut_gromp.mp4` - gromp DP nut trial compilation

### Task 2: Peg Insertion
- `peg_dp.mp4` - vanilla DP peg trial compilation
- `peg_gromp.mp4` - gromp DP peg trial compilation

### Task 3: Battery Cover Placement
- `remote_dp.mp4` - vanilla DP remote cover trial compilation
- `remote_gromp.mp4` - gromp DP remote cover trial compilation

### Task 4: USB Insertion
- `usb_dp.mp4` - vanilla DP USB trial compilation
- `usb_gromp.mp4` - gromp DP USB trial compilation

## File Format Guidelines
- Images: JPEG or PNG format
- Videos: MP4 format with H.264 codec for best browser compatibility
- Recommended video resolution: 720p or 1080p
